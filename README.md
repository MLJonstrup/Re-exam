## Opsætning & Installation

1. **Klon repository:**
   ```bash
   git clone https://github.com/MLJonstrup/Re-exam.git 
   ```

2. **Installer afhængigheder:**
   ```bash
   cd Re-exam
   npm install
   ```

4. **Kør appen:**
   ```bash
   npx expo start -tunnel
   ```
